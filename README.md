# 📊 Business Analytics Dashboard

A full-stack analytics platform that delivers real-time insights, predictive analysis, and data-driven decision support through an interactive and modern dashboard interface.

---

## 🚀 Features

* 📈 Interactive dashboards for business insights
* 🤖 Machine Learning-based predictions
* ⚠️ Automated warnings & anomaly detection
* 💡 Opportunity identification from data trends
* 📊 Data exploration and visualization
* 🔄 Real-time API integration

---

## 🧱 Tech Stack

### Frontend

* React.js
* CSS (Custom styling for modern UI)

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Cloud via MongoDB Atlas)

### Machine Learning API

* Python
* Flask
* Scikit-learn

---

## 📂 Project Structure

```
analytics-project/
│
├── frontend/       # React frontend
├── backend/        # Node.js + Express API
├── ml-api/         # Python ML service
├── .gitignore
├── package.json
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the **backend** folder:

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
ML_API_URL=http://localhost:8000
```

---

## 🧪 Run Locally

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo.git
cd analytics-project
```

---

### 2. Start Backend

```
cd backend
npm install
npm start
```

---

### 3. Start Frontend

```
cd frontend
npm install
npm start
```

---

### 4. Run ML API

```
cd ml-api
pip install -r requirements.txt
python app.py
```
